# @stacksjs/bun-queue-dashboard

A Vue-powered dashboard for monitoring and managing your bun-queue jobs.

## Installation

This is a private package used internally within the bun-queue monorepo. It is not published to npm.

## Usage

```bash
# Development
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

The dashboard provides a web interface for monitoring your queue system, accessible via the browser after starting the dev server.

## Features

- **Dashboard Overview** - Real-time summary of queue health and activity
- **Job Monitoring** - Track active, completed, and failed jobs
- **Metrics Visualization** - Charts and graphs powered by D3
- **Group Management** - View and manage job groups
- **Batch Tracking** - Monitor batch job progress
- **Job History** - Browse and inspect individual jobs
